Список основных тегов языка HTML

В наше время создать сайт довольно просто и для этого не надо знать все HTML-теги. Для начала стоит начать с самых популярных и часто используемых. Их хватит для создания примитивной страницы в интернете!

Напоминание

HTML состоит из тегов — команд, которые указывают браузеру, как отображать помещённый в них текст. Это и есть элементы веб-страницы.

Теги <html> , <head> и <body>

На каждой веб-странице есть три обязательных тега, они формируют структуру документа и содержат информацию для браузеров и поисковых систем.

<html> — контейнер для содержимого страницы.

<head> содержит служебную информацию. Здесь подключаются стили, указывается заголовок страницы, подключаются метатеги.

<body> содержит контент — всё, что отображается в браузере.

Тег <div>

Тег <div> — тег, определяющий логическое разделение содержимого веб-cтраницы. Теги div можно использовать для центрирования блоков содержимого, создания эффектов столбцов, создания различных областей цвета и многого другого.

<div class="content">
  <h2>Обо мне</h2>
  <p>Люблю изучать что-то новое.</p>
</div>

Тег <a>

Тег <a> является одним из важных элементов HTML и предназначен для создания ссылок. В зависимости от присутствия атрибутов name или href тег <a> устанавливает ссылку или якорь. Якорем называется закладка внутри страницы, которую можно указать в качестве цели ссылки. При использовании ссылки, которая указывает на якорь, происходит переход к закладке внутри веб-страницы.

<a href="URL">Обычная сылка</a>
<a name="идентификатор"> Ссылка с заглушкой</a>

Теги <ul> и <li>

<ul> создаёт неупорядоченный список, то есть элементы в нём не пронумерованы, а помечены маркером. Внутри тега с помощью <li> добавляются элементы списка.

<ul>
  <li>Первый элемент</li>
  <li>Второй элемент</li>
  <li>Третий элемент</li>
</ul>

Кстати, именно с помощью этих элементов была создана шапка этого сайта!

Тег <p>

Определяет текстовый абзац. Тег <p> является блочным элементом, всегда начинается с новой строки, абзацы текста идущие друг за другом разделяются между собой отбивкой. Величиной отбивки можно управлять с помощью стилей. Если закрывающего тега нет, считается, что конец абзаца совпадает с началом следующего блочного элемента.

<p> Тут должен быть ваш текст </p>

Тег <br>

Тег <br> переносит текст на другую строку, не начиная нового абзаца.
